# Signal | Trend Intelligence Platform

Signal is a real-time platform that tracks and analyzes global trends across the economy, climate, and technology. It helps you understand what's happening in the world by turning raw data into clear, visual signals.

## Key Features

- **Live Signal Feed**: Keep track of hundreds of data points from around the world as they happen.
- **Detailed Analysis**: Deep dive into any signal with interactive charts and historical trends.
- **Smart Tracking**: See which trends are rising, falling, or showing unusual patterns.
- **News Integration**: Get relevant news headlines alongside data to understand the "why" behind every change.
- **Mobile First**: A beautiful, responsive design that works perfectly on your phone, tablet, and computer.

## Getting Started

### 1. Installation
Clone the repository and install the dependencies:

```bash
npm install
```

### 2. Run Locally
Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the platform in action.
